# python-backend-test

Задание:<br/>
  Создать 3 страницы:<br />
	1. Главная с записями о ДДС, где можно фильтровать по дате, статусу, типу, категории и подкатегории.<br />
	2. Страница создания или редактирования записи, где есть форма данных и автоматические фильтры подкатегорий на основе категорий.<br />
	3. Страница для добавления/редактирования/удаления статусов, типов, категорий и подкатегорий, а также установка зависимостей.<br />

<br />Атрибуты таблицы:<br />
	1. Дата создания (создаётся автоматически, но может быть изменена), тип данных - дата;<br />
	2. Статус (Бизнес, Личное, Налог, ....., не обязательный пункт), тип данных - строчный;<br />
	3. Тип (Пополнение, Списание, .....), тип данных - строчный;<br />
	4. Категория (Инфраструктура, Маркетинг, .....), тип данных - строчный;<br />
	5. Подкатегория (Подкатегория может быть присвоена только своей категории), тип данных - строчный;<br />
	6. Сумма (в рублях, например: 1000р.), тип данных - числовой;<br />
	7. Комментарий (не обязательный, в свободной форме), тип данных - строчный.<br /><br />
  Ссылка на полное задание - https://drive.google.com/file/d/1vbPk2aiMe52pDFW57zMUDMaW7rqrHypc/view <br /><br />

<br />Инструкция к запуску:<br />
Данный код может быть запущен в любом Python интерпретаторе, который может работать с несколькими файлами (Например: VS Code, Pycharm). Чтобы запустить код необходимо перейти в папку с файлом 'manage.py' с помощью команды:<br />
  cd путь_к_папке<br />
и вписать в терминал команду:<br />
  python manage.py runserver<br />
Если возникла ошибка на отсутствие библиотек, прописать в терминал следующие команды:<br />
  pip install django<br />
  pip install djangorestframework<br />
После чего всё должно заработать.<br />
